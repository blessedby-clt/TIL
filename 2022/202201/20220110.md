## 새롭게 알게 된 내용

#### (1) 시계열

* 정상성 - 단위근 검정(unit test)
  * ADF test
  * ADF-GLS test
  * PP test
  * KPSS - 이 검정만 귀무가설이 시계열이 정상이다 임.
* 정규성 검정
  * Shapiro-Wilk test 등
* 자기상관
  * Ljung-Box test
  * Durbin-Watson test - 검정통계량 범위로 확인. 0에 가까울수록 양의 상관관계, 4에 가까울수록 음의 상관관계를 가짐
* 등분산성
  * Goldfeld-Qundt test - 귀무가설 : 등분산이다.
  * Breusch-Pagan test
  * Bartlett's test
